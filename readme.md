> open app link (intent, deeplink, universallink) in browsers

## Usage

CDN:

```html
<script src="https://unpkg.com/applync@latest"></script>
<script>
	applync('app://xxx');
</script>
```

AMD:

```javascript
require('applync')('app://xxx');
```
